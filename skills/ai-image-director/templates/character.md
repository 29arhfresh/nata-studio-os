# Template — Character Design

Use this template for original character concept art, character design sheets, illustration-style character development, game art, animated feature character studies, and concept art for non-photographic characters.

---

## When to Use This Template

- Original character design for games, animation, comics, or brand mascots
- Character concept art for pre-production development
- Character model sheets (turnaround views, expression sheets, proportion references)
- Fantasy, sci-fi, or genre character design
- Villain, hero, or supporting character concepts
- Non-human or creature design
- Brand mascot and illustrated character development

Use `portrait.md` when the subject is a photorealistic human. Use this template when the output is clearly illustrated, stylized, or in an animation/game aesthetic.

## Section 4: Costume and Equipment Design

### Costume Design Framework

**Silhouette first**: The costume must contribute to a readable silhouette. What large shape does the costume add?

**Function before decoration**: The costume should make sense for the character's world and role. A warrior's armor protects. A wizard's robes impede movement. Choices should be narratively motivated.

**Material language**: Materials communicate character. Worn leather = experience and practicality. Pristine plate armor = nobility or naivety. Patched cloth = poverty or resourcefulness. Expensive silk = wealth or deceit.

### Costume Template

```
[COSTUME CATEGORY — armor/robes/uniform/casual/ceremonial/survival gear].

[GARMENT 1 — type, material, color, condition, distinctive features].
[GARMENT 2 — type, material, color, condition, distinctive features].
[FOOTWEAR — type, material, height, condition].
[HEAD COVERING — if any: type, material, color].
[ACCESSORIES — belt, bag, hood, cloak, gloves, etc.].
[EQUIPMENT — primary weapon/tool/prop: material, condition, visual character].

Condition: [NEW AND PRISTINE / WELL-MAINTAINED / WORN AND WEATHERED / BATTLE-DAMAGED / MAKESHIFT AND REPAIRED].
Fit: [TAILORED AND FITTED / OVERSIZED AND LAYERED / LOOSE AND PRACTICAL / SKINTIGHT AND SLEEK].
Decoration: [NONE / MINIMAL FUNCTIONAL / SYMBOLIC MARKINGS / ELABORATE ORNAMENTATION].

## Section 6: Lighting for Character Design

Lighting in character design illuminates the design rather than sets a scene mood.

### Standard Character Design Lighting

**Model sheet lighting** (neutral and informative):
```
Neutral studio lighting, 5500K, even diffused, 
no strong directional shadows, 
character fully readable, no dramatic atmosphere, 
lighting to reveal design not create mood.
```

**Three-quarter dramatic** (reveals volume):
```
Three-quarter lighting from camera upper left, 
medium contrast, slight rim light from behind right, 
shadow side retains form detail, 
lighting reveals costume and face texture.
```

**Hero lighting** (for presentation art):
```
Dramatic hero lighting, strong key from slightly below and [DIRECTION], 
atmospheric quality, [GENRE-APPROPRIATE LIGHT COLOR — warm magic glow / cool technology blue / eerie green].
```

## Section 7: Presentation Formats

### Model Sheet (Multiple Views)

For consistency reference, request multiple views in one image:

```
Character model sheet layout, [CHARACTER DESCRIPTION].
Show: front view (0°), three-quarter view (45°), profile view (90°), 
back view (180°), [EXPRESSION SHEET if needed].
All views same scale, aligned at foot line, neutral pose, arms slightly away from body.
Model sheet format, white or pale grey background, clean production design document.
```

### Single Hero Pose

```
[CHARACTER DESCRIPTION].
Heroic three-quarter pose, [SPECIFIC ACTION OR STANCE — weapon raised / contemplative stance / dynamic mid-action].
[ENVIRONMENT HINT — implied, not detailed], [LIGHTING — genre appropriate].
Character concept illustration, [STYLE REFERENCE], ultra-high resolution, 
production quality character art.
```

### Expression Sheet

```
[CHARACTER DESCRIPTION — establish design in first sentence].
Expression sheet showing six expressions on the same character:
1. Neutral / resting
2. Happy / laughing
3. Angry / threatening
4. Sad / defeated
5. Surprised / shocked
6. Determined / focused
Each expression bust-up (chin to above crown), evenly spaced, 
same character design across all six, identical costume details.
Model sheet format, white background, animation production quality.
```

## Negative Prompt (Character Design)

```
blurry, low quality, inconsistent design, multiple characters in same image 
(unless model sheet), photographic realism (unless specified), 
generic fantasy stock character, poor anatomy, 
clothing clips through body, floating accessories, 
poorly drawn hands, face distortion, off-model features
