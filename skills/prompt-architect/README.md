# Prompt Architect

A skill for designing, refining, and optimizing prompts for AI models — text, image, video, code, and research contexts.

## Overview

Prompt Architect helps you craft high-quality prompts that get better results from AI systems. Whether you're writing a simple instruction or a complex multi-step workflow, this skill provides frameworks, patterns, templates, and checklists to guide your process.

## Files

| File | Purpose |
|------|---------|
| `SYSTEM_PROMPT.md` | Master system prompt that activates the Prompt Architect persona |
| `WORKFLOW.md` | Step-by-step process for building prompts from scratch |
| `FRAMEWORKS.md` | Reusable prompt engineering frameworks (RISEN, COSTAR, etc.) |
| `PATTERNS.md` | Common prompt patterns and anti-patterns |
| `CHECKLIST.md` | Quality checklist before finalizing a prompt |
| `EXAMPLES.md` | Annotated real-world examples across domains |
| `templates/` | Ready-to-fill templates by use case |

## Templates

| Template | Use Case |
|----------|----------|
| `basic-prompt.md` | Simple single-turn instructions |
| `advanced-prompt.md` | Multi-step, role-based, or chain-of-thought prompts |
| `image-prompt.md` | Image generation (Midjourney, Stable Diffusion, DALL-E, etc.) |
| `video-prompt.md` | Video generation and editing prompts |
| `coding-prompt.md` | Code generation, review, and debugging prompts |
| `research-prompt.md` | Research, summarization, and analysis prompts |

## Quick Start

1. Identify your **goal** — what output do you want?
2. Choose the right **template** from `templates/`
3. Apply a **framework** from `FRAMEWORKS.md`
4. Run through the **checklist** in `CHECKLIST.md`
5. Test, evaluate, and iterate

## When to Use This Skill

- You're getting inconsistent or low-quality AI outputs
- You want to create reusable prompt templates for your team
- You're building a system prompt for an AI agent or assistant
- You need to adapt a prompt for a different model or modality
- You want to learn prompt engineering best practices
