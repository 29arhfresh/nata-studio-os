# Website Builder Skill

A skill for planning, generating, and refining websites — from landing pages to full business sites.

## What this skill does

- Gathers project requirements (purpose, audience, content, style)
- Recommends the right site type and template
- Generates structured page layouts with copy placeholders
- Produces semantic HTML/CSS scaffolds or framework-specific components
- Reviews and iterates until the site is ready to ship

## When to use this skill

- User asks to "build a website", "create a landing page", or "make a portfolio"
- User provides a brand brief, product description, or business overview
- User wants to turn a Figma/mockup into working code
- User needs a fast, production-ready starting point for a web project

## Templates included

| Template | Best for |
|---|---|
| `landing-page.md` | Single-page marketing or product launches |
| `portfolio.md` | Freelancers, designers, and developers |
| `business-site.md` | Multi-page company or service websites |

## Quick start

1. Ask the user for: site purpose, target audience, key pages, and any brand assets.
2. Pick the closest template from `templates/`.
3. Follow `WORKFLOW.md` step by step.
4. Use `CHECKLIST.md` before delivering the final output.
