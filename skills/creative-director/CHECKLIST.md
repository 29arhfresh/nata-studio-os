# Checklist — Creative Director

Use this checklist at each stage of the creative production pipeline. Every item must be confirmed before advancing to the next stage.

---

## Stage 1: Brief Completion

- [ ] Brand name and core promise documented
- [ ] Campaign or project objective written in one sentence (≤ 300 characters)
- [ ] Target audience defined: age range, geography, income level, interests
- [ ] Tone descriptors listed (minimum 3, maximum 8; all lowercase)
- [ ] Deliverable list confirmed: formats, aspect ratios, quantities
- [ ] Visual references named and explained (minimum 2)
- [ ] Mandatory brand elements listed (colors, logos, taglines)
- [ ] Forbidden elements listed (visual styles, motifs, or associations to avoid)
- [ ] Timeline per deliverable documented

---

## Stage 2: Concept Approval

- [ ] Concept stated in one sentence (subject + tension + resolution)
- [ ] Narrative arc named and confirmed (transformation / aspiration / revelation / contrast / intimacy)
- [ ] Central image described in concrete visual terms (subject, action, environment, lighting, emotion)
- [ ] Three reference directions identified with individual rationales
- [ ] Concept aligns with the strategic objective from the brief
- [ ] Concept reviewed and approved by decision-maker before advancing

---

## Stage 3: Visual System Sign-Off

### Color Strategy
- [ ] Primary color defined with hex value and zone assignment
- [ ] Accent color defined with hex value and focal point application
- [ ] Neutral defined with hex value and usage rule
- [ ] WCAG AA contrast verified for any text-on-background combination
- [ ] Palette reviewed against mandatory and forbidden color constraints from brief

### Typography System
- [ ] Primary typeface named with display size minimum
- [ ] Secondary typeface named with body readability at 14px confirmed
- [ ] Tertiary typeface named (or explicitly absent) with usage constraint
- [ ] Typeface pairing validated for contrast of category (not weight alone)

### Composition and Lighting
- [ ] Composition principle assigned per deliverable type
- [ ] Lighting signature documented (source, quality, direction, temperature)
- [ ] Visual system consistent with brand references in brief

---

## Stage 4: Art Direction Completeness

For each deliverable:

- [ ] Deliverable type, aspect ratio, resolution, and color space specified
- [ ] Composition spec names the rule and positions the subject in the frame
- [ ] Lighting spec includes source, direction, Kelvin temperature, and modifiers
- [ ] Color application names palette values and the zones where each appears
- [ ] Typography application specifies typeface, scale relationship, and tracking intent (where applicable)
- [ ] Mood statement connects this deliverable to the campaign concept in one sentence
- [ ] No vague language ("beautiful", "nice", "interesting") appears in any directive
- [ ] Technical notes capture any production-specific constraint

---

## Stage 5: Creative Review

For each completed deliverable:

- [ ] Brand Alignment scored 1–10 with one-sentence rationale
- [ ] Composition Quality scored 1–10 with one-sentence rationale
- [ ] Color Consistency scored 1–10 with one-sentence rationale
- [ ] Storytelling Clarity scored 1–10 with one-sentence rationale
- [ ] Technical Execution scored 1–10 with one-sentence rationale
- [ ] Weighted total calculated (Brand 30% / Composition 20% / Color 20% / Story 15% / Tech 15%)
- [ ] Grade assigned: Exceptional / Strong / Acceptable / Needs Revision / Reject
- [ ] All revision directives from previous cycles addressed (if applicable)
- [ ] No dimension scoring below 5
- [ ] Deliverable verified against brief mandatory elements and forbidden elements
- [ ] Final approval status documented: Approved / Revise / Rejected

---

## Cross-Campaign Consistency Audit

Run this audit when two or more deliverables are produced for the same campaign:

- [ ] All deliverables use only palette values from the approved color strategy
- [ ] Typography hierarchy is consistent across all deliverables
- [ ] Composition principle applied correctly per deliverable type
- [ ] Lighting signature is recognisable across all image and video deliverables
- [ ] Brand voice in copy aligns with tone descriptors in the brief
- [ ] Character appearance (if applicable) is consistent across image and video
- [ ] No deliverable could be mistaken for a competitor's campaign
